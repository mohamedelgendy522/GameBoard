Detailed Description

Represents the Ultimate Tic-Tac-Toe game board.

This class inherits from the generic Board<char> and implements the rules of the Ultimate X-O game, including move validation, win/draw detection, and board display logic.

Parameters

  1. Identify which 3×3 sub-board the last move belongs to.

  4. Check if Matrix[][] has a 3-in-a-row → Ultimate victory.

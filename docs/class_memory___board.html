Detailed Description

Represents the board for Memory Tic-Tac-Toe game.

This class inherits from the generic Board<char> class and implements the classic Tic-Tac-Toe logic but is designed for memory-based gameplay where symbols are hidden during display.

Note
Board size is fixed at 3x3 (standard Tic-Tac-Toe)
Warning
Symbols are stored normally but displayed as 'H' (hidden)

Detailed Description

Represents a 5×5 board for the Big X-O Tic-Tac-Toe game.

Handles input/output operations for the Big Tic-Tac-Toe game.

The board:

  • Uses a 5×5 grid initialized with '.'
  • Validates moves
  • Counts sequences of identical symbols
  • Detects win / draw / end-game states
See also
Board

Responsibilities:

  • Creating players
  • Requesting user moves
  • Supporting human and computer interactions

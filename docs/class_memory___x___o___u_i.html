Detailed Description

User Interface class for Memory Tic-Tac-Toe game.

Inherits from the generic UI<char> base class and provides specific functionality for Memory Tic-Tac-Toe where symbols are displayed as 'H' (hidden) instead of actual 'X' or 'O'.

\doxysubsection{Detailed Description}
User Interface class for the \doxylink{class_pyramid}{Pyramid} X-\/O game. 

This class inherits from {\ttfamily \doxylink{class_u_i}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} and provides Pyramid-\/specific player registration, move collection, and display handling.

It ensures players follow the spatial constraints of the pyramid layout when selecting moves.

\doxysubsection{Detailed Description}
Represents the board used in the Tic-\/\+Tac-\/\+Toe game. 

This board stores the X and O symbols placed by players and implements:\+
\begin{DoxyItemize}
\item \doxylink{class_move}{Move} application logic
\item Win checking
\item Draw checking
\item Game termination conditions
\end{DoxyItemize}

A move history queue is also maintained to track the order of moves.

\doxysubsection{Detailed Description}
User Interface class for Memory Tic-\/\+Tac-\/\+Toe game. 

Inherits from the generic {\ttfamily \doxylink{class_u_i}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} base class and provides specific functionality for Memory Tic-\/\+Tac-\/\+Toe where symbols are displayed as \textquotesingle{}H\textquotesingle{} (hidden) instead of actual \textquotesingle{}X\textquotesingle{} or \textquotesingle{}O\textquotesingle{}.

\doxysubsection{Detailed Description}
User Interface class for the Numerical X-\/O (Tic-\/\+Tac-\/\+Toe) game. 

Inherits from the generic {\ttfamily \doxylink{class_u_i}{UI}\texorpdfstring{$<$}{<}int\texorpdfstring{$>$}{>}} base class and provides Numerical-\/\+X-\/\+O–specific functionality for player setup and move input.

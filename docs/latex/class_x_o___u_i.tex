\doxysubsection{Detailed Description}
User Interface class for the X-\/O (Tic-\/\+Tac-\/\+Toe) game. 

Inherits from the generic {\ttfamily \doxylink{class_u_i}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} base class and provides X-\/\+O–specific functionality for player setup and move input.

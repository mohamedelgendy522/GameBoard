\doxysubsection{Detailed Description}
Represents the \doxylink{class_pyramid}{Pyramid} X-\/O game board. 

This class inherits from the generic {\ttfamily \doxylink{class_board}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} class and implements the logic required for the \doxylink{class_pyramid}{Pyramid} X-\/O variant, including special move rules, win-\/condition evaluation, and draw detection.

The \doxylink{class_pyramid}{Pyramid} version typically uses a non-\/standard board arrangement (e.\+g., triangular or progressively expanding rows), and the class handles all movement validation and game-\/state updates accordingly.

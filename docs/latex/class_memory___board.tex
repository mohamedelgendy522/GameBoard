\doxysubsection{Detailed Description}
Represents the board for Memory Tic-\/\+Tac-\/\+Toe game. 

This class inherits from the generic {\ttfamily \doxylink{class_board}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} class and implements the classic Tic-\/\+Tac-\/\+Toe logic but is designed for memory-\/based gameplay where symbols are hidden during display.

\begin{DoxyNote}{Note}
\doxylink{class_board}{Board} size is fixed at 3x3 (standard Tic-\/\+Tac-\/\+Toe) 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Symbols are stored normally but displayed as \textquotesingle{}H\textquotesingle{} (hidden) 
\end{DoxyWarning}

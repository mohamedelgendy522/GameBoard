\doxysubsection{Detailed Description}
Represents the board for Misère Tic-\/\+Tac-\/\+Toe (Inverse X-\/O) game. 

This class inherits from the generic {\ttfamily \doxylink{class_board}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} class and implements the specific logic required for Misère Tic-\/\+Tac-\/\+Toe, where the player who makes three in a row LOSES the game.

\begin{DoxyNote}{Note}
\doxylink{class_board}{Board} size is fixed at 3x3 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Uses minimax algorithm with alpha-\/beta pruning for AI moves 
\end{DoxyWarning}

\doxysubsection{Detailed Description}
Represents the board for Diamond Tic-\/\+Tac-\/\+Toe game. 

This class inherits from the generic {\ttfamily \doxylink{class_board}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} class and implements a diamond-\/shaped board (10x10 matrix with valid positions forming a diamond). The winning condition requires both a line of 3 and a line of 4 in different directions.

\begin{DoxyNote}{Note}
\doxylink{class_board}{Board} size is 10x10 with only 25 valid positions forming a diamond shape 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Uses minimax algorithm with depth limitation and alpha-\/beta pruning 
\end{DoxyWarning}

\doxysubsection{Detailed Description}
Represents the \doxylink{class_ultimate}{Ultimate} Tic-\/\+Tac-\/\+Toe game board. 

This class inherits from the generic {\ttfamily \doxylink{class_board}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} and implements the rules of the \doxylink{class_ultimate}{Ultimate} X-\/O game, including move validation, win/\+draw detection, and board display logic.

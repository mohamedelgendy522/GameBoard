\doxysubsection{Detailed Description}
Represents the SUS game board. 

Inherits from the generic {\ttfamily \doxylink{class_board}{Board}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} class and implements the SUS-\/specific logic, including move updates, score calculation, win/\+lose/\+draw detection, and tracking row/\+column/\+diagonal patterns.

\doxysubsection{Detailed Description}
User Interface class for the \doxylink{class_ultimate}{Ultimate} X-\/O game. 

Inherits from the generic {\ttfamily \doxylink{class_u_i}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} and provides functions specific to setting up players and retrieving moves for the \doxylink{class_ultimate}{Ultimate} Tic-\/\+Tac-\/\+Toe experience.

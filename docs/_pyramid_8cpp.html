map<pair<int,int>,bool> check

A map used to mark which board positions are valid playable cells in the Pyramid X-O structure.

Valid cells = true Blocked/invalid cells = false

Example:

  • Levels 0–2 are blocked: '#'
  • Level 3: middle 3 cells playable
  • Level 4: all 5 cells playable

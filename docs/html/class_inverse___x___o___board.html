Detailed Description

Represents the board for Misère Tic-Tac-Toe (Inverse X-O) game.

This class inherits from the generic Board<char> class and implements the specific logic required for Misère Tic-Tac-Toe, where the player who makes three in a row LOSES the game.

Note
Board size is fixed at 3x3
Warning
Uses minimax algorithm with alpha-beta pruning for AI moves

Detailed Description

Declares classes for the 5×5 Big Tic-Tac-Toe (X-O) game.

Defines the Tic-Tac-Toe board and UI classes.

This header contains:

  • X_O_Board: Implements move validation, 5×5 grid logic, sequence counting, win/draw detection.
  • XO_UI: Handles player creation and move input/output for the Big X-O game.

These classes extend the generic framework defined in BoardGame_Classes.h.

This file provides:

  • Infinity_Board: A board implementation for the Tic-Tac-Toe game.
  • Infinity_UI: The user interface responsible for player creation and move input.

Both classes extend the generic framework defined in BoardGame_Classes.h.
